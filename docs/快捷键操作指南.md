# Online Notes Platform 快捷键操作指南

本文档汇总了平台支持的所有快捷键，旨在帮助用户通过键盘高效完成笔记的编辑、管理与协作。

## 1. 全局通用快捷键

| 功能 | Windows / Linux | macOS | 说明 |
| :--- | :--- | :--- | :--- |
| **保存笔记** | `Ctrl` + `S` | `Cmd` + `S` | 立即保存当前编辑的内容，触发自动摘要生成 |
| **全屏模式** | `Ctrl` + `Shift` + `F` | `Cmd` + `Shift` + `F` | 切换编辑器至沉浸式全屏模式 |
| **退出全屏** | `Esc` | `Esc` | 退出全屏模式 |

## 2. 编辑器格式化快捷键

在笔记编辑器（Tiptap）中，支持以下快捷键快速调整文本样式：

| 功能 | Windows / Linux | macOS | 说明 |
| :--- | :--- | :--- | :--- |
| **粗体** | `Ctrl` + `B` | `Cmd` + `B` | 加粗选中的文本 |
| **斜体** | `Ctrl` + `I` | `Cmd` + `I` | 倾斜选中的文本 |
| **下划线** | `Ctrl` + `U` | `Cmd` + `U` | 给选中文本添加下划线 |
| **插入链接** | `Ctrl` + `K` | `Cmd` + `K` | 弹出链接插入对话框 |
| **撤销** | `Ctrl` + `Z` | `Cmd` + `Z` | 撤回上一步操作 |
| **重做** | `Ctrl` + `Y` <br> `Ctrl` + `Shift` + `Z` | `Cmd` + `Shift` + `Z` | 恢复撤回的操作 |

## 3. Markdown 语法快捷键 (输入即生效)

编辑器支持标准的 Markdown 输入习惯，输入特定字符后按 `Space` (空格) 即可自动转换格式：

| 功能 | 输入指令 | 示例 |
| :--- | :--- | :--- |
| **一级标题** | `#` + `Space` | 输入 `# 标题` -> 变大标题 |
| **二级标题** | `##` + `Space` | 输入 `## 标题` -> 变二级标题 |
| **三级标题** | `###` + `Space` | 输入 `### 标题` -> 变三级标题 |
| **无序列表** | `-` + `Space` 或 `*` + `Space` | 输入 `- 项目` -> 变圆点列表 |
| **有序列表** | `1.` + `Space` | 输入 `1. 项目` -> 变数字列表 |
| **引用块** | `>` + `Space` | 输入 `> 引用` -> 变引用样式 |
| **代码块** | ` ``` ` + `Enter` | 输入 ` ``` ` 后回车 -> 插入代码块 |
| **水平分割线** | `---` + `Enter` | 输入 `---` 后回车 -> 插入分割线 |

## 4. 协作与辅助功能

| 功能 | Windows / Linux | macOS | 说明 |
| :--- | :--- | :--- | :--- |
| **打开评论** | `Alt` + `C` | `Option` + `C` | 打开/关闭右侧评论侧边栏 |
| **搜索确认** | `Enter` | `Enter` | 在搜索栏输入关键词后，按回车执行搜索 |
| **发送 AI 消息** | `Enter` | `Enter` | 在 AI 宠物对话框中发送消息 (Shift+Enter 换行) |

## 5. 常见问题

- **快捷键不生效？**
  - 请确保输入法处于英文状态（尤其是 Markdown 语法输入）。
  - 某些浏览器插件可能会占用特定快捷键，尝试禁用插件或更换浏览器测试。
- **全屏模式无法退出？**
  - 如果 `Esc` 键失效，请尝试将鼠标移动到屏幕顶部中间，点击浏览器弹出的 "退出全屏" 提示，或再次按下 `Ctrl+Shift+F`。
